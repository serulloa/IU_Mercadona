MEMORIA FINAL - INTERFACES DE USUARIO GRUPO 9

ESTUDIO DE LA COMPETENCIA

Nuestra investigación arranca con el análisis de dos tipos de plataformas:

  • El rey de las plataformas de compra online: Amazon.
  • Plataformas de supermercados: Carrefour, Hipercor y Tesco.
Pensamos que lo ideal es una combinación de patrones de diseño utilizados en los dos tipos de plataformas. Amazon es una de las principales web de compra online de la que está más familiarizada las personas. En cuanto supermercados, hemos encontrado buenos ejemplos que nos han sido muy útiles a la hora de pensar el diseño de cada sección de nuestra web.

A continuación mostramos un pequeño análisis de los pros y contras que hemos encontrado en las distintas plataformas:

Pros y Contras

Amazon

PROS

  • Su buscador resalta mucho conduciendo al usuario a utilizarlo
  • Elementos de compra como el carrito bien situados proporcionando un control absoluto
  • Selección de categorías muy buena con la grán cantidad de tipos de productos que ofrece
  • Recomendaciones al usuario con ofertas y promociones para vender más al usuario
  • Proporciona muchas opciones para el filtrado de los productos
  • Productos bien definidos
  • Contiene una sección fija que conduce a la finalización de la compra

CONTRAS

  • Login al que le falta un Deep Background
  • La estética general de Amazon no es muy llamativa ni agradable para el usuario
  • No tiene Breadcrumbs

Carrefour

PROS

  • Estética muy buena al igual que como muestran el contenido
  • Cuando recorremos todos los productos al final tienen una sección para ver más productos o cargar siempre automáticamente
  • Feature, Search and Browse para facilitar la experiencia de navegación
  • Flat Menus para las distintas categorías de productos
  • Breadcrumbs para ubicar al usuario en la página web
  • Grid of Equals para los productos
  • Se conduce a la finalización al usuario, orientandole en el proceso de compra

CONTRAS

  • Cabecera mal estructurada, con tres columnas visibles con diferentes alineamiento
  • No sigue las regla 7 +/- 2 elementos para reducir la carga de memoria

Hipercor

PROS

  • Feature, Search and Browse para la navegación
  • Breadcrumbs para ubicar al usuario en la página web
  • Grid of Equals para los productos

CONTRAS

  • Menú no desplegable, un click más para seleccionar la subcategoría
  • Cabecera mal estructurada
  • Accediendo a una categoría en especial los productos destacados van al final, poco visible para el usuario

Tesco

PROS

  • Estilo sencillo y ordenado
  • Contrasting Font Weights para resaltar elemntos más importantes sobre otros
  • Breadcrumbs para ubicar al usuario en la página web

CONTRAS

  • Página que respeta pocos patrones de diseño
  • El usuario realiza muchos clicks al buscar un producto por categoría
  • Menú no desplegable, un click más para seleccionar la subcategoría
  • Feature, Search and Browse no es utilizado correctamente, tiene barra de búsqueda pero no existe una sección con productos principales u ofertas
